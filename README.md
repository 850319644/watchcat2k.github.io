## 博客更新

把 `.md` 文件放入 `post` 文件夹，然后按照示例文件加上头信息即可。

## 问题总结

在头信息里，日期 `date` 字段的时分秒应保持为0，否则本地无法预览该文件。不过 push 到 github 之后，即使日期 `date` 字段的时分秒不为0，也是正常的。
